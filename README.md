# Ingredients

- [Node.js](https://nodejs.org) or [Bun](https://bun.dev)
- [Vue](https://vuejs.org)
- [Nuxt](https://nuxt.com)

# Setup

```bash
bun install
```
or
```bash
npm install
```

# Development Server

```bash
bun run dev
```
or
```bash
npm run dev
```

# Production

```bash
bun run build
```
or
```bash
npm run build
```
